# Chromium Audio Equalizer

This folder contains an unpacked Chromium extension generated by `eq.py`.

## Load it in Chromium

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this `chromium_equalizer` folder

## Notes

- Chromium extensions cannot run Python in the browser UI, so the live equalizer logic is implemented with JavaScript and the Web Audio API.
- `eq.py` exists to generate and refresh the extension files from Python.
- The equalizer uses 10 peaking filters centered at standard frequencies from 31 Hz to 16 kHz.
- Some protected pages like the Chrome Web Store do not allow content scripts, so the equalizer will only work on normal sites.
