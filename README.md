## How to Use
1. Create a folder named "10-band EQ" and paste all the files in this folder.
2. Open `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked**
5. Select this '10-band EQ' folder

## Notes
- The equalizer uses 10 peaking filters centered at standard frequencies from 31 Hz to 16 kHz.
- Some protected pages like the Chrome Web Store do not allow content scripts, so the equalizer will only work on normal sites.
